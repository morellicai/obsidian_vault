# Conceito Geral de Função

Uma **Função** é definida como uma **relação especial**. Nessa relação, **todo elemento** de um conjunto de partida, chamado **domínio** ($D$), associa-se a um **único elemento** de um conjunto de chegada, chamado **contradomínio** ($CD$).

- **Notação:** Uma função $f$ é representada como $f: A \to B$.
    - $A$ é o domínio.
    - $B$ é o contradomínio.
    - A notação $y = f(x)$ significa que $y$, pertencente ao contradomínio, é a **imagem** ($Im$) de $x$.
- **Exemplo:** Se $f(x) = 2x$ associa $A = {0, 1, 2, 3}$ a $B = {0, 2, 3, 4, 5, 6, 7}$, o domínio é $D = A$ e o contradomínio é $CD = B$. A imagem será $Im = {0, 2, 4, 6}$.

## Função Polinomial

Uma função polinomial é toda função dada pela forma $P(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n$.

- **Coeficientes:** Os termos $a_i$ são os coeficientes de $i = 0, 1, 2, \ldots, n$. 
- **Grau:** $n$ representa o grau de $P(x)$.
- **Exemplos:**
    - $P(x) = 2x - 3$: Coeficientes $a_0 = -3$ e $a_1 = 2$, com grau 1.
    - $P(x) = x^2 - 3x + 2$: Coeficientes $a_0 = 2, a_1 = -3$ e $a_2 = 1$, com grau 2.
- O material inclui a determinação das **raízes** (o valor de $x$ tal que $P(x) = 0$) e o esboço do gráfico da função.

## Função Racional

Uma função racional é toda função dada pela forma $\mathbf{R(x) = \frac{P(x)}{Q(x)}}$, onde $P(x)$ e $Q(x)$ são funções polinomiais, e o denominador **$Q(x)$ deve ser diferente de zero**.

- **Raízes:** As raízes de $R(x)$ são os valores de $x$ no domínio onde $P(x) = 0$; elas nem sempre existem.
- **Exemplo (Assíntotas):** Para $R(x) = \frac{1}{x^2 - 1}$, o domínio é $\mathbb{R} - {-1, 1}$, pois o denominador não pode ser zero. Neste exemplo, não há raízes, pois $P(x) = 1$ para todo $x$ no domínio.
- O conceito de **Assíntota Vertical** é introduzido. Para o exemplo, $x = -1$ e $x = 1$ são assíntotas verticais, com o valor de $R(x)$ tendendo a $+\infty$ ou $-\infty$ à medida que $x$ se aproxima desses valores.

## Função Logarítmica

A função logarítmica é toda função dada pela forma $\mathbf{f(x) = \log_a x}$, onde a base $a$ deve satisfazer a condição $\mathbf{0 < a \neq 1}$.

- **Domínio, Contradomínio e Imagem:**
    - $D = \mathbb{R}^*$ (reais positivos).
    - $CD = \mathbb{R}$.
    - $Im = \mathbb{R}$.
- **Raízes:** A raiz de uma função logarítmica sempre ocorre em $\mathbf{x = 1}$, pois $\log_a x = 0 \implies x = a^0 \implies x = 1$.
- **Crescimento/Decrescimento:**
    - A função é **crescente** se $a > 1$.
    - A função é **decrescente** se $0 < a < 1$.
- **Exemplo:** $f(x) = \log_{10} x$ é crescente, pois a base $a=10 > 0$.

## Função Exponencial

A função exponencial é toda função dada pela forma $\mathbf{f(x) = a^x}$, onde a base $a$ também deve satisfazer $\mathbf{0 < a \neq 1}$.

- **Domínio, Contradomínio e Imagem:**
    - $D = \mathbb{R}$.
    - $CD = \mathbb{R}$.
    - $Im = \mathbb{R}^*$ (reais positivos).
- **Raízes:** **Não existem raízes**, pois $a^x$ é sempre maior que zero ($\mathbf{a^x > 0}$ SEMPRE).
- **Crescimento/Decrescimento:**
    - A função é **crescente** se $a > 1$.
    - A função é **decrescente** se $0 < a < 1$.
- **Exemplo:** $g(x) = 0.2^x$ é decrescente, pois a base $a=0.2$, e $0 < a < 1$.

## Funções Trigonométricas

O material apresenta as três funções trigonométricas principais, enfatizando seus domínios, imagens e periodicidade:

### A. Função Seno ($f(x) = \operatorname{sen}(x)$)

- É uma função **periódica de período $2\pi$**, ou seja, $\operatorname{sen}(x + 2\pi) = \operatorname{sen}(x)$ para todo $x$ real.
- $D = \mathbb{R}$.
- $CD = \mathbb{R}$.
- $Im = [-1, 1]$.
- **Raízes:** $\operatorname{sen}(x) = 0 \implies x = 0 + k\pi$, onde $k$ pertence a $\mathbb{Z}$ (conjunto dos números inteiros).

#### B. Função Cosseno ($f(x) = \cos(x)$)

- É uma função **periódica de período $2\pi$**, ou seja, $\cos(x + 2\pi) = \cos(x)$ para todo $x$ real.
- $D = \mathbb{R}$.
- $CD = \mathbb{R}$.
- $Im = [-1, 1]$.
- **Raízes:** $\cos(x) = 0 \implies x = \pi/2 + k\pi$, onde $k \in \mathbb{Z}$.

#### C. Função Tangente ($f(x) = \operatorname{tg}(x)$)

- É uma função **periódica de período $\pi$**, ou seja, $\operatorname{tg}(x + \pi) = \operatorname{tg}(x)$ para todo $x$ pertencente ao domínio de $f$.
- **Domínio:** O domínio é restrito por causa das assíntotas verticais: $D = {x \in \mathbb{R} : x \neq \pi/2 + k\pi, k \in \mathbb{Z}}$.
- $CD = \mathbb{R}$.
- $Im = \mathbb{R}$.
- **Raízes:** $\operatorname{tg}(x) = 0 \implies x = 0 + k\pi$, onde $k \in \mathbb{Z}$.
